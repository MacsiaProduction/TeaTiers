## Вывод по Gemini

Gemini API доступен из ЕС без привязки карты, так что «Gemini free-tier» **может быть основным** (Gemini 3.5 Flash с бесплатным лимитом). По условиям Google вы сами владеете сгенерированным текстом и можете его сохранять, и исключений для коммерческого использования нет. Однако бесплатная «unpaid» квота **используется для обучения Google** (то есть «тренирует» модель), что приемлемо для неконфиденциальных названий чая. Качество на русском и китайском ожидается очень высоким благодаря мощному многоязычному корпусу Google. 

Поэтому **рекомендуем** оставить план: *Gemini free-tier как первичный*, YandexGPT Lite как запасной. Gemini обеспечит лучший баланс качества и бесплатности, а Yandex – страховой вариант (низкое время отклика в РФ).

## Конфигурация запросов для Gemini

Для основного сервиса следует использовать модель **Gemini 3.5 Flash** (ID `gemini-3.5-flash`) через API Google AI. Формируем запрос в формате JSON с описанием задачи; можно задать схему вывода (JSON Schema) для структурированных ответов (Gemini API поддерживает структурированный вывод). Пример: 
```
POST https://us-central1-aiplatform.googleapis.com/v1beta1/projects/PROJECT_ID/locations/us-central1/models/gemini-3.5-flash:generateText
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

{
  "prompt": {
    "context": "Программа должна перевести китайское название чая в корректную транслитерацию на русский и добавить пиньинь:",
    "messages": [
      {"role":"user","content":"大红袍"}
    ]
  },
  "temperature": 0.2,
  "top_p": 0.95,
  "max_output_tokens": 64,
  "schema": {
    "type": "object",
    "properties": {
      "chinese": {"type": "string"},
      "pinyin": {"type": "string"},
      "russian": {"type": "string"}
    },
    "required": ["chinese","pinyin","russian"]
  }
}
```
Gemini позволяет отключить логирование обучения только при оплате (платный сервис). В бесплатном режиме запретить использование данных Google нельзя, но это допустимо для небезопасных данных. 

**Порядок fallbacks:** первично — Google Gemini, при проблемах (превышение лимитов, недоступность VPN) — YandexGPT Lite, затем — дополнительные опции (DeepSeek как «чрезвычайный» zh-бустер, Groq/Mistral для экспериментов).

## Источники

- Google Gemini API (официальная документация): подтверждает доступность free-tier в Европе.  
- Google Gemini API Terms: «Google не претендует на авторство сгенерированного контента»; «для Unpaid квоты Google использует данные для улучшения моделей».  
- DeepSeek Terms: все права на Outputs принадлежат пользователю; DeepSeek сообщает о использовании данных (есть опция «Improve model» с возможностью отключения).  
- DeepSeek Free Tier (PricePerToken): 5 млн бесплатных токенов при регистрации, без карты.  
- Groq API (документация): свободная регистрация без карты, стабильный free-tier (например, Llama-3.3-70b: 30 RPM, 1 000 RPD, 12 000 TPM).  
- Groq Services Agreement: по умолчанию Groq **не хранит** входные/выходные данные дольше нужного и **не использует их для обучения** без явного разрешения.  
- Mistral AI Blog: введён бесплатный уровень на «Plateforme» с полной изоляцией и опцией «zero-retention» (исключительно для прототипирования, с гарантиями приватности).  

## Неясности / что не удалось проверить

- **Gemini:** нет явных публичных цифр RPM/TPM для бесплатной квоты в официальных источниках (нужен вход в Google Cloud).  
- **Формулировка ТЗ:** точные ограничения free-tier Google (10 RPM, 250k TPM) берутся из неофициальных источников, требуют подтверждения.  
- **YandexGPT:** отсутствуют общедоступные документы по использованию и хранению пользовательских данных в бесплатной версии, поэтому базируемся на практике (ранее подтверждено возможность хранения вывода).  
- **Качество вывода:** конкретные примеры русской транслитерации сравнить трудно без реальных тестов; оценки сделаны на основе конструкции моделей (подробные тесты не найдены).
