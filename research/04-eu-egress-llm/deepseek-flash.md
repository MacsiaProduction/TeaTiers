Основной вывод этого исследования однозначен: **Gemini в бесплатном тарифе не может использоваться в ЕС** для коммерческих продуктов. Это связано не с техническими ограничениями VPN, а с прямым запретом в пользовательском соглашении (ToS) для разработчиков, обслуживающих пользователей из ЕС.

Поэтому план "Gemini primary, YandexGPT fallback" отменяется. Новая стратегия: **YandexGPT** как основной (прямой доступ из РФ, без VPN, разрешено хранение вывода), а **DeepSeek** в режиме "частотного резервирования" через VPN (DE) для русского и китайского языков.

Ниже представлен детальный анализ, обосновывающий это решение.

### 💎 Ключевой вывод по Google Gemini

Использование Gemini API Free из ЕС невозможно для коммерческого проекта согласно ToS.

*   **Запрет в ToS**: Согласно разделу "Use Restrictions" в Gemini API Additional Terms of Service (от 20 мая 2025 г.), вы можете использовать только **платные услуги**, когда предоставляете доступ к API-клиентам пользователям в **Европейской экономической зоне, Швейцарии или Великобритании** (ЕС).
*   **Причина**: Бесплатный уровень API Gemini исторически не был доступен в ЕС по юридическим причинам, и это ограничение не было снято. Это не технический блок IP-адресов (VPN не поможет), а прямое нарушение условий использования.
*   **Важное уточнение**: VPN из Германии (ЕС) обеспечивает обход географических блокировок, но не снимает юридических обязательств по ToS. Предоставление услуги пользователям из РФ с сервера в ЕС может потребовать соблюдения законов ЕС (например, GDPR). Поставщики услуг могут требовать платную подписку для обслуживания пользователей из ЕС, и эта информация может отсутствовать в общих условиях.

### 🛠️ Рекомендации по выбору и конфигурация

#### Стратегия TeaTiers
Выбран путь **"частичного резервирования"**:

*   **Основной LLM**: **YandexGPT** (надежность, прямой доступ, соблюдение ToS, отличное качество).
*   **Резервный LLM (приоритет)**: **DeepSeek** (высокое качество ru/zh, низкая цена, доступен через VPN).
*   **Вторичный резерв**: **Groq** (высокая скорость при отказе DeepSeek).

#### Детальная конфигурация API

**1. YandexGPT (Основной)**
*   **Модель**: `yandexgpt-lite` (для баланса скорости и стоимости).
*   **Эндпоинт**: `https://llm.api.cloud.yandex.net/llm/v1alpha/chat/completions`
*   **Структурированный вывод**: `"temperature"`: 0.1; продуманный `system_prompt` и `instruction` для JSON.
*   **Примечание**: Используйте API‑ключ или IAM‑токен; отслеживайте квоты.

**2. DeepSeek (Резервный, китайский/русский)**
*   **Модель**: `deepseek-chat`.
*   **Эндпоинт**: `https://api.deepseek.com/v1` (OpenAI‑совместимый).
*   **Структурированный вывод**: `"response_format": { "type": "json_object" }` (`response_format.type`).
*   **Примечание**: Дайте инструкцию `system_prompt` о строгом JSON. Доступен через OpenRouter с 5M токенов.

**3. Groq (Вторичный резерв, высокая скорость)**
*   **Модель**: `llama-3.1-8b-instant`.
*   **Эндпоинт**: `https://api.groq.com/openai/v1` (OpenAI‑совместимый).
*   **Структурированный вывод**: `"response_format": { "type": "json_object" }` (`response_format.type`).
*   **Примечание**: Ключ через консоль; для `strict` требуется схема.

**4. Миграция с Gemini (если необходимо)**
*   Используйте OpenRouter (`openrouter.ai`) или Google Cloud с оплатой.
*   Модель: `gemini-1.5-flash`.
